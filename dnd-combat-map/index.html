            Tutorial

            Welcome to the D&D Combat Board. This tool lets you manage a grid-based combat map, tokens, initiative, objects, and character inventories.

            Main features

            • Place, move, rotate, and delete tokens on the grid.
            • Track initiative order and turn progression.
            • Create objects with presets, color, shape, and size.
            • Use the inventory system for characters, equipment, item counts, and gold.
            • Use the Items page as a central allowed-item storage.

            How it connects

            • The legend contains controls for grid settings and quick reference.
            • The Inventory modal manages each character's items and gold.
            • The Items modal defines what item names can be added to characters.
            • The All Characters Inventory modal shows shared items and total gold for all characters.
            • Export/import keeps inventory data backed up, while browser storage preserves it automatically.

            Controls

            • Legend is always visible in compact form when minimized.
            • Tutorial opens this guide from the legend header.
            • ESC closes any open modal.
