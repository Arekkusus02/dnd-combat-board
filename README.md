# dnd-combat-board
a simple dnd combat board. here you can roll and track initiatice,
location and direction of each participant in combat and can edit
the board's height and lenght based on you preferences. simple,
but useful. Created with the power of GitHub Copilot and ChatGPT,
they helped a lot.

to run it, simply double click the html file and select a browser of your choice
everything else is explained in the legend, have a wonderful adventure.
